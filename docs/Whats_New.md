# What's New

This page records significant updates to the ETIS publication, framework, website, teaching ecosystem, repositories, and supporting resources.

## July 2026

### Initial Public Release (Version 1.0)

The first public release of the ETIS publication, website, and supporting ecosystem is available.

Highlights:

- Complete online publication
- Four-part ETIS framework
- Thirty-nine chapters
- Eight appendices
- Volume I and Volume II navigation
- Framework reference materials
- Teaching with ETIS resources
- Resource Center
- ETIS publication website

Future updates, resources, repositories, and ecosystem components will be announced here.
