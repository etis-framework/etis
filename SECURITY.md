# Security Policy

## Security Scope

The ETIS repository is primarily a publication, documentation, and educational-resource repository.

It does not provide hosted software services, production APIs, or software products that process user data.

## Reporting Security Issues

If you believe you have discovered a security-related issue affecting:

- The ETIS website
- Repository infrastructure
- GitHub Pages deployment
- Publication assets
- Repository configuration

please contact the repository owner privately before public disclosure.

## Supported Versions

| Version | Supported |
|----------|-----------|
| 1.x | Yes |
| < 1.0 | No |

## Responsible Disclosure

Please provide:

- Description of the issue
- Reproduction steps
- Potential impact
- Suggested remediation (if known)

The repository owner will review reports and determine appropriate corrective action.

## Publication Content

Errors, typographical issues, broken links, figure issues, or technical corrections are not considered security vulnerabilities and should be reported through normal publication-feedback channels.

## Disclaimer

This repository is provided for educational, research, and publication purposes. No warranty is provided regarding the suitability of any material for a particular purpose.
